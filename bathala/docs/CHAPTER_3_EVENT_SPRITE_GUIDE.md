# Chapter 3 Event Sprite Guide

This file lists the Chapter 3 educational events that need dedicated event sprites or illustrations.

Chapter theme: `The Skyward Citadel`
Visual direction: floating skies, sacred mountains, dreamlike indigenous imagery, celestial spirits, Mindanao cultural motifs.

- Time: `Day`
- What the image should show:
  A mythic creation scene where sky and sea mirror each other, with islands suspended in a dreamlike expanse and the Sarimanok as a central symbol.
- Key visual elements:
  - Floating islands or mirrored world composition
  - Vast sky-ocean emptiness
  - Sarimanok flying across the scene
  - Strong mythic creation atmosphere
- Simple art brief:
  `A surreal origin-world with floating islands and a Sarimanok flying through a sky that feels like the world is still being formed.`
- AI prompt:

Keep the pixel art clean, readable, and visually strong with simple shapes and a limited color palette. Make the Sarimanok distinct with ornate bird features, long decorative tail feathers, bright Maranao-inspired color accents, and a mythical fortune-bird identity rather than a generic fantasy bird. Emphasize mythic origin imagery, airy space, and strong readable composition. Keep it uniform with the game's existing pixel art look, no text, no UI, transparent or simple self-contained background suitable for event illustration use.`
