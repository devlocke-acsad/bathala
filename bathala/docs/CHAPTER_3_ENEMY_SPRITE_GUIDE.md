# Chapter 3 Enemy Sprite Guide

This file lists the Chapter 3 enemies that need dedicated art direction and AI-ready prompts.

Chapter theme: `The Skyward Citadel`
Visual direction: floating sky realms, celestial shrines, divine birds, storm spirits, sacred mountain imagery, simple readable pixel art.

All prompts below follow these rules:
- `1:1 aspect ratio`
- `simple pixel art`
- `same art style as the game Bathala`
- `not overly detailed`
- `clean readable silhouette`
- `limited color palette`
- `no text`
- `no UI`

Each enemy includes 2 prompt types:
- `Splash art` = more dramatic character-focused illustration
- `Simple bg` = same character but with a light, simple background suitable for game asset generation
